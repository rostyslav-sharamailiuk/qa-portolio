# Bug Report Example 1

**Title:** Checkout fails with valid credit card  
**Environment:** Chrome 117, Windows 10  
**Steps to Reproduce:**  
1. Add product to cart  
2. Proceed to checkout  
3. Enter valid credit card info  
4. Click Pay  
**Expected Result:** Payment successful  
**Actual Result:** Error message "Payment failed"  
**Severity:** High
